The ``1-my_list`` module
==========================

Using ``MyList``
-----------------

Import the class::

    >>> MyList = __import__('1-my_list').MyList

Create and use a MyList instance::

    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Empty list::

    >>> empty = MyList()
    >>> empty.print_sorted()
    []

With negative numbers, appended out of sorted order::

    >>> neg_list = MyList()
    >>> neg_list.append(3)
    >>> neg_list.append(-1)
    >>> neg_list.append(0)
    >>> neg_list.append(-5)
    >>> print(neg_list)
    [3, -1, 0, -5]
    >>> neg_list.print_sorted()
    [-5, -1, 0, 3]
    >>> print(neg_list)
    [3, -1, 0, -5]
